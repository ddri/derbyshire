# derbyshire
A little studio assistant to help me with things like reverb and delay times etc.

## Current Features
- Tempo-based timing calculator (reverb/delay times for note values)
- Pitch/tempo relationship calculators for sampling workflow
- Modern glassmorphism UI with responsive design
